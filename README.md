## Usage

To get started with this project, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone git@github.com:RunSabba/terraformproject.git
    cd terraformproject
    ```

2. **Initialize Terraform**:
    ```sh
    terraform init
    ```

3. **Apply the configuration**:
    ```sh
    terraform apply
    ```
